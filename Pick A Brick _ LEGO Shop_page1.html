Basic Search
Here, you can search for bricks and elements by general categories, features, and colors – or see all your options at once.

Search by Category
Find bricks and elements by themes and categories (such as Accessories, Fences & Ladders, Transportation, and many others). Select a category from the drop-down menu to see all the bricks and elements that match your model – in all shapes and colors. 

Search by Color Family
Choose a color family (such as Reds, Yellows, or Greens) from the drop-down menu to see a list of all the bricks and elements in that color family.

    Advanced Search
    Here, you can search for bricks and elements by specific details.

    Search by Brick Name
    If you know the name of a specific LEGO brick or element (for example, Brick 1X2) or part of the name (for example, 1X2), type it into the Brick Name field and click the blue button to display a list of all bricks that match your description.

    Search by Element ID
    Each specific LEGO brick and element has an Element ID – a unique number that you can find at the back of all building instructions from January 2006 onwards. Enter this number (for example, 4496343) into the Element ID field and click the blue button to find your specific brick or element.

    Search by Design ID
    The Design ID is a number that identifies the shape of the brick or element, and can be found inside all LEGO bricks. Enter this number (for example, 42022) into the Design ID field and click the blue button to see a list of all bricks or elements in that design.

    Search by Exact Color
    Choose a specific color (such as Dark Red, Transparent Blue, or Bright Yellow) from the drop-down menu to see a list of all the bricks and elements in that exact color.
